Conway's Game of Life

Rules of the game:

  1. Any live cell with fewer than two live neighbors dies, as if by under population.
  2. Any live cell with two or three live neighbors lives on to the next generation.
  3. Any live cell with more than three live neighbors dies, as if by overpopulation.
  4. Any dead cell with exactly three live neighbors becomes a live cell, as if by reproduction.

The simulation below was created for the MagmaHackers challenge. The Start button begins the simulation, while the Pause button pauses the simulation, which can be resumed by either pressing Start or Resume. The Reset button resets the population randomly. For this simulation the population is randomly set, for now it can't be set manually.
